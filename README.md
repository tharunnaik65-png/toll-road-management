# Toll Road Management - Full Stack

This workspace contains a full-stack Toll Road Management System prototype.

Backend (Express + MongoDB): c:\Users\tharu\OneDrive\Desktop\toll road mangement\backend

Frontend (React + Vite + Tailwind): c:\Users\tharu\OneDrive\Desktop\toll road mangement\frontend

Quick start

1. Backend

```
cd backend
npm install
cp .env.example .env
# adjust MONGO_URI and JWT_SECRET as needed
npm run dev
```

2. Frontend

```
cd frontend
npm install
# optionally set VITE_API_URL in .env
npm run dev
```

The backend seeds an admin user on first start: admin@example.com / password
